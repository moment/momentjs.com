---
title: Year
version: 1.0.0
signature: |
  moment().year(Number);
  moment().year(); // Number
  moment().years(Number);
  moment().years(); // Number
---


Gets or sets the year.

Accepts numbers from -270,000 to 270,000.

**2.16.0** deprecated using ``moment().years()``. Use ``moment().year()`` instead.
